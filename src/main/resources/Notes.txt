Существует4 модификатора доступа (access modifiers)
- public (используются в любом месте программы)
- private (используются только внутри класса)
- protected (используются только в тех классах,которые наследуются)
- default (package-default) (доступные в рамках пакета,в котором лежит данный класс)

Применяются только к переменным и методам класса

Геттеры и сеттeры работают в паре с private модификатором доступа.
Они испоьзуются, когда переменные чувствительны к правильным значениям.
Переменную нужно превратить в метод.
Сеттеры не выкидывают значение, поэтому void.
Геттеры викидывают значение, поэтому устанавливается тип возвращаемого значения int, String и т.д.

Static метод хранится только внутри класса и не хранится внутри переменной класса.
Используется,когда необходимо обратитьсяк переменной не создавая экземпляр класса, а также, чтобы не забивать память лишними переменными.
В основном используется для создания счетчика.

Конструктор - метод, который всегда срабатывает, когда объявляется экземпляр класса после ключевого слова New.
В конструкторе нет возвращаемого значения и название метода совпадает с названием класса.
Запускается самостоятельно без нашего участия.
Можно добавлять параметры.
К нему также можно применять различные модификаторы доступа.

Все что вызывается из какого объекта называется методом, а не функцией.
